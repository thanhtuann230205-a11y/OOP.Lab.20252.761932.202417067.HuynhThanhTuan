If you create a constructor method to build a DVD by title then create a constructor method to build a DVD by category. Does JAVA allow you to do this?
1. Nếu title và category cùng kiểu dữ liệu và chỉ có 1 tham số thì java không cho phép làm như vậy.
2. Nếu title và category không cùng kiểu dữ liệu hoặc khác số lượng tham số hoặc khác thứ tự các tham số thì java sẽ cho phép làm như vậy.
Ví dụ cho 1:
public DVD(String title){}
public DVD(String category){}
-> java không cho phép.
Ví dụ cho 2:
public DVD(int title){}
public DVD(String category){}
-> java cho phép.
public DVD(String title,int id){}
public DVD(String category){}
-> java cho phép.
public DVD(String title,int id){}
public DVD(int id,String category){}
-> java cho phép.
